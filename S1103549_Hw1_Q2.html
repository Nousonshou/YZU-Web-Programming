Q2: How to create a registration form using HTML and CSS, please explain in your own words.



Explanation

I started by setting the html head, title and body first.

Since the <h> tag was setting too much space between the elements I opted for <span>, <legend>, <span>and <label>. I then set the input field by simply using the <input> tag and adjusting its attributes as well as the other tags' to obtain the effect I wanted.

I then opted for a <fieldset> to make the different sections of the form more clearly distinguisheable.

The hardest part was then setting the CSS side of the file, which definetely looks cleaner compared to html, but my trial and error approach made the process unclear and time consuming. I added the class attribute to the html element so that they could be modified by the classes I created in CSS in order to modify the layout, the color, the font and the transparency of both the text and the background.

Instead of using "opacity" in CSS, which was making things harder by applying transparency to both the background picture and the text content, I opted for setting rgba for the background color and abandoned the idea of setting up a picture as the .main's background. In the end the result gave a minimalistic look and was satisfying.

Because the source code is unclear I apologize to the user and the developers' community. I will commit on clearing it up and improve it; I was considering adding a background picture slide show as a final touch.
