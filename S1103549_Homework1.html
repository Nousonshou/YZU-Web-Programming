This form is optimized for Edge and Chrome.

To create this form I started with a simple html form, an example of which can be found on my domain at webprogramming.bitninja.me/form. A few differences are made such as the Title, Check marks, and drop down menu being added as well as the submit button.

The next step is to add the css. First the background for the page, then the background for the header of the registration form, then finally for the registration form itself.

The final step is to add the css to each element. This consists of things such as color, positioning, spacing, and even hover behavior.
